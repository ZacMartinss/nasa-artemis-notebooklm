# 🚀 NASA Artemis e o Futuro da Exploração Lunar

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como parte do desafio prático da DIO utilizando a ferramenta NotebookLM.

O objetivo é construir um caderno temático sobre a exploração espacial, com foco na NASA, no Programa Artemis, na futura base lunar e na história das missões tripuladas que ajudaram a humanidade a alcançar a Lua.

Através do NotebookLM, serão utilizadas fontes confiáveis para criar uma base de conhecimento capaz de auxiliar nos estudos, revisões e aprofundamento dos temas relacionados à exploração espacial.

---

## 🎯 Objetivos de Estudo

- Compreender a história da exploração espacial.
- Estudar as missões Apollo e seus impactos.
- Entender o funcionamento do Programa Artemis.
- Conhecer os planos da NASA para uma presença humana permanente na Lua.
- Explorar os desafios tecnológicos da construção de uma base lunar.
- Investigar como as missões lunares servirão de preparação para futuras viagens a Marte.
